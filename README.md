# 📊 YouTube Channel Data Analysis (Origami Tutorials)

## 🎯 Project Objective
The goal of this project is to analyze two years of real-world data from my personal Origami YouTube Channel (550+ subscribers, 120k+ total views). I wanted to understand viewer behavior, find the best days to upload content, and identify which topics drive the most traffic.

## 🛠️ Tools Used
- **Python** (Data Cleaning & Analysis)
- **Pandas** (Data Manipulation)
- **Matplotlib & Seaborn** (Data Visualization)
- **Regex (`re`)** (Text processing and string manipulation)

## 💡 Key Insights Discovered
1. **The Weekend Spike:** Data reveals a massive spike in viewership on **Saturdays** compared to the rest of the week. *Actionable Insight:* Scheduling new video publishes for Friday evenings or Saturday mornings will maximize the initial algorithm push.
2. **Seasonal Dominance:** The top-performing videos are all holiday-themed (Christmas Santa, Halloween Bat, etc.). Event-driven content yields the highest Return on Time Invested (ROTI).
3. **Algorithm Impact:** Analyzed the relationship between Impressions and Views to understand how YouTube's recommendation algorithm distributes my content.

## 🔒 Data Privacy Note
Due to data privacy best practices (GDPR compliance), the raw CSV files for this dataset are not included in this public repository. Only the analysis scripts (`youtube_analysis_py.py`) and aggregated visual insights are shared below to ensure reproducibility without compromising private account data.
